# Оформление отчёта 8П в Microsoft Word (по аналогии с примером `нир.docx`)

Файл `нир.docx` в корне репозитория — **бинарный**; его стиль можно воспроизвести в Word после импорта текста.

## Рекомендуемый порядок работ

1. Откройте пример `нир.docx` и зафиксируйте: шрифт основного текста, кегль, межстрочный интервал, поля страницы, оформление заголовков (Уровень 1–3).
2. В Word: **Вставка → Объект→ текст из файла** или скопируйте содержимое `Otchet_NIR_8P_Efremova_V_N.md` через редактор с предпросмотром (например, Typora / VS Code) и вставьте в документ с применёнными стилями.
3. Замените подчёркивания на титульном листе на ФИО научного руководителя и полное название вуза.
4. Вставьте **рисунки** из `docs/experiment_practicum/out/` (ряд, MAE, hold-out, WAPE) в раздел 5 или в приложение; подписи: «Рисунок N — …».
5. Сгенерируйте **оглавление**: Ссылки → Оглавление → автоматическое.
6. Нумерация страниц: снизу по центру или справа (как в методичке).

## Перенос из Markdown

- Уравнения \(R_t\) при необходимости замените встроенным редактором формул Word.
- Блоки `mermaid` Word не отрисует: сохраните диаграммы как изображения (экспорт из GitHub / mermaid.live) и вставьте как «Рисунок».

## Zotero

1. Импортируйте `Экспортированные записи.txt` (уже в библиотеке пользователя).
2. **Файл → Импорт** → выберите `Zotero_dopolneniya_8P.ris` (дополнительные источники [14–18] из отчёта).
3. В документе Word подключите плагин Zotero и замените скобочные ссылки вида [6] на поля цитирования по стилю кафедры (ГОСТ / APA).
